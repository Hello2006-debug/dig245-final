DC NAVIGATOR

My project, DC NAVIGATOR, is an interactive navigation map tool. Inspired by the challenges of location buildings,and campus resources and hidden gems, the goal was to create a map that not only reflects the building but adds details on the resource they offer.
The design process began with Figma prototypes to establish the layout and user flow. I focused on simple user centered design to ensure that the interface was cozy,welcoming and reflecting Davidson College.
The app is built with HTML, CSS, and JavaScript, incorporating features like Leaflet and GeoJson Ultimately, this project aligns with the course objectives by allowing me as a student to put in practice all the topics we have covered throughout the semester. I hope it serves as a valuable tool and I hope to develop it further outside the classroom for users.

Features:
  • Locate campus buildings and facilities effortlessly.
  • Filter locations by categories such as academic, dining, or student life.
  • Discover detailed information about each location.

Stay tuned for exciting updates, including user-contributed places and interactive building views!
